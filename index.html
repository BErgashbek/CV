3. HTML nima?

HTML-Hyper Text Markup Language.

HTML Web-sahifalarni yaratish uchun standart belgili tildir.

HTML Web-sahifaning strukturasi.

HTML ko'plab elementlardan iborat.

HTML elementlari brauzerga kontentni qanday ko'rsatishni aytadi.

HTML elementlari go'yo brauzerga "Bu sarlavha", "Bu paragraf", "Bu havola" deb kontent qismlarini aytib turadi
